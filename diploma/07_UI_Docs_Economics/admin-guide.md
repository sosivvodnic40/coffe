# Инструкция администратора

## Вход в систему

1. Откройте http://localhost:5173/admin
2. Введите учётные данные:
   - Email: `admin@cappuccino.kz`
   - Пароль: из файла `.env` (по умолчанию `Cappuccino2025!`)
3. Нажмите «Войти».

## Панель управления

### Статистика
- **Всего броней** — общее количество в системе
- **Ожидают** — брони со статусом pending
- **Подтверждено сегодня** — confirmed на текущую дату
- **Гостей сегодня** — суммарное число гостей

### Управление бронями

1. Используйте фильтры: Все / Ожидает / Подтверждена / Отменена / Завершена
2. Для смены статуса выберите новое значение в выпадающем списке
3. Нажмите «Обновить» для перезагрузки данных

### Статусы броней

| Статус | Значение |
|--------|----------|
| pending | Новая заявка, ожидает обработки |
| confirmed | Бронь подтверждена администратором |
| cancelled | Бронь отменена |
| completed | Гость посетил ресторан |

## API для администратора

Альтернативно можно работать через REST API (см. `diploma/05_DB_API/api-spec.md`).

## Безопасность

- Не передавайте пароль третьим лицам
- Меняйте пароль в `.env` перед эксплуатацией
- Регулярно делайте резервную копию `data/cappuccino.db`
