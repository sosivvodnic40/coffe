# Инструкция пользователя

## Назначение

Сайт ресторана «Cappuccino» предназначен для:
- просмотра информации о заведении;
- изучения меню и акций;
- онлайн-бронирования стола.

## Запуск

1. Откройте браузер.
2. Перейдите по адресу: http://localhost:5173

## Навигация

| Раздел | Описание |
|--------|----------|
| О нас | История и описание ресторана |
| Меню | Блюда по категориям с ценами в тенге |
| Акции | Специальные предложения |
| Галерея | Фотографии интерьера |
| Резервация | Форма бронирования |
| Контакты | Адреса, телефоны, часы работы |

## Бронирование стола

1. Перейдите в раздел «Резервация».
2. Выберите локацию (Кабанбай Батыр или Аль-Фараби).
3. Заполните имя, телефон, дату, время, количество гостей.
4. Нажмите «Подтвердить резервацию».
5. Сохраните код брони (формат CP-XXXXXX).
6. Администратор подтвердит бронь по телефону.
