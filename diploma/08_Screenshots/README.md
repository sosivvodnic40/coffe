# Скриншоты для дипломного архива

Положите сюда PNG-файлы перед сборкой ZIP-архива.  
Рекомендуемое разрешение: 1920×1080 или 1440×900.

## Обязательный список

| Файл | Что снять |
|------|-----------|
| `01-home.png` | Главная страница (hero-блок) |
| `02-menu.png` | Раздел «Меню» с категориями |
| `03-promotions.png` | Раздел «Акции» |
| `04-reservation-form.png` | Форма бронирования |
| `05-reservation-success.png` | Экран с кодом CP-XXXXXX |
| `06-reservation-lookup.png` | Проверка брони по коду |
| `07-admin-login.png` | Страница входа /admin |
| `08-admin-dashboard.png` | Список броней и статистика |
| `09-admin-status.png` | Смена статуса брони |
| `10-npm-test.png` | Результат команды `npm test` |
| `11-api-health.png` | Браузер: localhost:3001/api/health |

## Как сделать

1. Запустите `npm run dev`
2. Откройте http://localhost:5173
3. Используйте Win+Shift+S (Windows) для скриншотов
4. Сохраните файлы в эту папку с именами из таблицы

## Для пояснительной записки

Вставьте скриншоты в раздел «Руководство пользователя» (главы 3–4).
